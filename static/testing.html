This is the testing page, any role can be selected to simulate playing as that role.
Use the God role if you want to enter the game without interfering with other players

Some key passages:
"Control: Variables": Controls all non user or faction related variables
"Control: Variables (users)": Controls all user related variables
"Control: Variables (factions)": Controls all faction related variables
"Control: Change passage": allows you to send other roles to passages
"Control: side passages": allows you to send other roles to side passages
"Control :lookup": allows you to change which user is playing which role
"Control :Full Reset": will reset the game to its base state and also delete the current user's history
